# The-royal-game-of-ur

This is a school web project which consisted in reproducing the game "royal game of ur" using js, html and css.
You can choose the number of pieces, activate the effect of square 8 and take turns playing.
At the end of the game, when a player has no more pieces in play, the score is displayed and the buttons are disabled. Player 1 has blue pieces, player 2 red.
